# Temperature Conversion Program 🌡️

This is my submission for **Task 1** of the Prodigy InfoTech Software Development Internship. 

## 📝 Description
A Python command-line application that converts temperatures between Celsius, Fahrenheit, and Kelvin. The user simply inputs a temperature value and its original unit, and the program calculates and displays the converted values for the other two units.

## 🚀 How to Run
1. Ensure you have Python installed on your system.
2. Open your terminal or command prompt.
3. Navigate to the folder containing this file.
4. Run the script using the following command:
   ```bash
   python main.py
